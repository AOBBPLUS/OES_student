该文件夹内包含的是学生前端所使用的api接口

banner是和幻灯片有关的接口，主要是获取所有幻灯片
comment是和评论有关的接口，主要有读取评论列表和添加评论
course是和课程有关的接口，主要有读取课程列表和读取课程详细信息
index中有首页面读取课程和教师信息的接口
login是登录相关的接口，主要有获取用户信息和读取输入信息以及微信登录
order是订单以及支付有关的接口，主要有生成订单，根据订单查询信息和生成二维码
register是和注册有关的接口，主要有注册和发送短信
teacher是和教师有关的接口，主要有读取教师列表和读取教师详细信息
vod是和视频播放有关的接口，主要有获取视频信息的内容
